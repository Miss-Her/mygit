添加修改文件：
1，添加文件到暂存区里： git  add 文件名
2，把所有在暂存区的文件都提交到仓库 ： git commit -m "注释"
3，检测是否有文件还未提交 ： git status
4,  查看文件修改了什么内容 ： git diff 文件名
5，查看历史记录 ： git log --pretty=oneline
版本回退：
1，回到上个版本 ：git  reset  --hard HEAD^
2，回到新版本 ：git reset  --hard 版本号（git  reflog）
3，丢弃（撤销）工作区的修改 ： git checkout -- 文件名
删除远程仓库的文件或目录：
删除a目录下的2.txt文件 ：git rm -r --cached a/2.txt 
删除a目录  ：git rm -r --cached a
git commit -m "删除a目录下的2.txt文件" 
git push
用-r参数删除目录, git rm --cached a.txt 删除的是本地仓库中的文件，
且本地工作区的文件会保留且不再与远程仓库发生跟踪关系，
如果本地仓库中的文件也要删除则用git rm a.txt

1,本地仓库和远程仓库关联：
git remote add origin https://github.com/Miss-Her/mygit.git
提交到远程：git push origin master

2,远程提交到本地：
git clone https://github.com/Miss-Her/testgit

出现问题：git remote rm origin  或者删除 .git/config/[remote "origin"]
然后再执行上面
